Zen Filter

Component Name: ZenFilterComponent.
Module Name: Zen Core

Introduction

The Zen Filter component used to display filters in various places, such as storefront product lists and admin-side list pages.

Sample Images




Filter Properties

1. @Input() filterDetails
It is a required property to show filter data. It contains some basic properties like
title - Filter title.
type - Filter type (e.g, radio, checkbox).
keyToEmit- Filter output name.
and etc.
The remaining properties are explained in detail in filter types.
Example:

//Ts
filterDetails: ZenFilterDetails[] = [
{ 
title:  'Choose Fruit',
type:  ZenFilterTypes.RADIO,
keyToEmit:  'choosenFruit', 
disabled:  false,
options: [
{value:  'orange', label:  'Orange', isDefault:  false, disabled:  false},
{ value:  'apple', label:  'Apple',  isDefault:  false, disabled:  false }
]
}]

//Html
<zen-filter  [filterDetails]="filterData" (onFilterChange)="onFilterChange($event)"></zen-filter>

2. @Input() filterHeader
It is not required, but we can customize our filter headers using the below properties.
hideHeader - It is used to hide filter header.
hideFilterHeading - It is used to hide filter heading.
heading - It is used to change filter heading name.
resetAllIconToolTip- It is used to change reset all icon tooltip name.
closeIconToolTip - It is used to change close icon tooltip name.
hideFilterCount - It is used to hide filter count.
emitWithOutButton - It is used to emit filter details without button.
filterSelectedMessage - It is used to change filter selected messages.
showSelectedFilter - It is used to selected filter label in chip.
applyButtonText - //It is used to change apply button text
saveButtonText - //It is used to change save filter button text
resetIconToolTip - //It is used to change reset icon name in expansion

//Ts
filterHeader: ZenFilterHeaders = { heading: 'My Filter', resetIconToolTip: 'Refresh filter', closeIconToolTip: 'Close filter', hideFilterCount: false, emitWithOutButton: false, showSelectedFilter: true, filterSelectedMessage: Filter Selected', hideHeader: false, hideFilterHeading: false}
//Html
<zen-filter  [filterDetails]="filterData"  [filterHeader]="filterHeader"  (onFilterChange)="onFilterChange($event)"></zen-filter>

3. @Input() expansionProperty
It is used to change the expansion panel design. Its properties are mentioned below.
expandMultiple - used to open multiple expansion panels.
expandAllByDefault - used to expand all expansion panels by default.
hideExpansion - used to hide the expansion panel.
hideFilterReset- used to hide the refresh icon in the expansion panel.
hideExpansionBorder - used to hide expansion panel border.
expansionWithBoxShadow - used to display expansion with box shadow.

Type 1:

//ts
expansionProperty: ZenFilterexpansionProperty = {
    expandMultiple:  false,
    expandAllByDefault:  false,
    isPanelHidden:  false,
    hideFilterReset:  false,
    hideExpansionBorder:  false,
    expansionWithBoxShadow:  false
}
//html
<zen-filter  [filterDetails]="filterData"  [filterHeader]="filterHeader"  (onFilterChange)="onFilterChange($event)" [expansionProperty]="expansionProperty"></zen-filter>

Type2:

4. @Output() filterClosed = new EventEmitter();
This property emits boolean value when filter tab is closed.

5. @Output() onFilterChange = new EventEmitter();
This property emits selected filter details.
eg:

{
"isPriceRange": {
    "fromRange": 30,
    "toRange": 50
},
"choosenFruit": "apple",
"selectedSize": "L",
"selectCheckBox": {
    "isPant": true,
    "isShirt": false,
    "isTshirt": false
}
}

6. @Input() savedFilterDetails
This input is used to show the saved filter option on zen filter. Its properties are mentioned below.
You may get queries for zba-shared library.
eg:
import { GET_ALL_SAVED_FILTERS, CREATE_SAVED_FILTERS, CHECK_DUPLICATE_FITER_NAME, DELETE_SAVED_FILTER } from “@zba/shared/queries/zen-filter.query”

Properties:

  1. getApiDetails - Input for getting saved filter details.
  2. createApiDetails - Input for creating a new saved filter.
  3. deleteApiDetails - Input to delete a single filter.

Model: SavedFilterDetails

eg:

savedFilterDetails: SavedFilterDetails = {
  getApiDetails: {
	query: GET_ALL_SAVED_FILTERS, //get query for saved filter,
	inputDetails: {
	  input: {
		storeId:  //storeid,
		code: //filtercode(ie,'ORDER_FILTER')
	  }
	}
  },
  createApiDetails: {
	query: CREATE_SAVED_FILTERS, //create query for saved filter,
	inputDetails: {
	  input: {
		storeId:  //storeid,
		code: //filtercode(ie,'ORDER_FILTER')
	  }
	}
  },
  deleteApiDetails: {
	query: DELETE_SAVED_FILTER,  //delete query for saved filter,
	inputDetails: {
	  input: {
		storeId:  //storeid,
		code: //filtercode(ie,'ORDER_FILTER')
	  }
	}
  },
  checkDuplicateFilterName: {
	query: CHECK_DUPLICATE_FITER_NAME, //delete query for saved filter,
	inputDetails: {
	  input: {} //no input required for this
	}
  }
}
   

##  Filter Types

## 1. Radio:

**Properties:**
title: string;                 -> title name of radio group
type: '**RADIO**	';                -> filter type
keyToEmit: string;				 -> field name where the output will be assigned there
disabled: boolean;         -> disable whole option group
options:[
{value: string | number, label: string, isDefault: boolean, disabled: boolean},
{value: string | number, label: string, isDefault: boolean, disabled: boolean},
etc...
]
**Code Example:**

    //ts
    filterData = [
		{
		title: 'Choose Fruit',
		type: ZenFilterTypes.RADIO,
		keyToEmit: 'fruitName',
		disabled: false,
		options: [
		{value: 'orange',label:'Orange',isDefault: false,disabled:false},
		{value: 'apple',label:'Apple',isDefault: true,disabled:false},
		]
		}
	}
	//html
	<zen-filter  [filterDetails]="filterData" (onFilterChange)="onFilterChange($event)"></zen-filter>

## 2. Checkbox:

**Properties:**
title: string;                  -> title name of checkbox group
type: '**CHECKBOX**';         -> filter type
keyToEmit: string;				 -> field name where the output will be assigned there
disabled: boolean;         -> disable whole checkbox group
checkbox:[
{ label: string, keyToEmit: string, isDefault: boolean, disabled: boolean},
{ label: string, keyToEmit: string, isDefault: boolean, disabled: boolean},
etc...
]
**Code Example:**

    //ts
    filterData = [
		{
		title: 'Shirt Size',
		type: ZenFilterTypes.CHECKBOX,
		keyToEmit: 'shirtSize',
		disabled: false,
		checkbox: [
		{keyToEmit: 'isL',label:'L',isDefault:false,disabled:false},
		{keyToEmit: 'isXl',label:'Xl',isDefault:true,disabled:false},
		{keyToEmit: 'isXxl',label:'Xxl',isDefault:false,disabled:false}
		]
		}
	]
	//html
	<zen-filter  [filterDetails]="filterData" (onFilterChange)="onFilterChange($event)"></zen-filter>

## 3. Textbox:

**Properties:**
title: string;                  -> title name of textbox
type: '**TEXT**';                  -> filter type
keyToEmit: string;				 -> field name where the output will be assigned there
fieldLabel: string         -> textbox label name
disabled: boolean;         -> disable textbox
minLength: number    -> validator for minimum length
maxLength: number   -> validator for maximum length

**Code Example:**

    //ts
    filterData = [
		{
		title:  'Search Name',
		type:  ZenFilterTypes.TEXT,
		keyToEmit:  'searchName',
		fieldLabel:  'Enter Name',
		minLength:  2,
		maxLength:  5,
		disabled:  false
		}
	]
	//html
	<zen-filter  [filterDetails]="filterData" (onFilterChange)="onFilterChange($event)"></zen-filter>

## 7.  Basic slider
**Properties:**
title: string;                  -> title name 
type: '**SLIDER**';    	     -> filter type
keyToEmit: string;				 -> field name where the output will be assigned there
disabled: boolean;         -> disable textbox
 slider: {
	type: 'basic',    -> slider type
	value: number, -> slider default value
	minRange: number, -> slider min value
	maxRange: number, -> slider max value
	currencyUnit?: 'USD' -> if you want to display currency symbol near value then it can be used
}

**Code Example:**

    //ts
    filterData = [
		{
		title:  'Choose size',
		type:  ZenFilterTypes.SLIDER,
		keyToEmit:  'size',
		disabled:  false,
		slider: {
			type: 'basic',
			value: 5,
			minRange: 1,
			maxRange: 100
		} as  ZenFilterSlider
		}	
	]
	//html
	<zen-filter  [filterDetails]="filterData" (onFilterChange)="onFilterChange($event)"></zen-filter>

## 8.  Date Picker
**Properties:**
title: string;                  -> title name 
type: '**DATEPICKER**';    	     -> filter type
keyToEmit: string;				 -> field name where the output will be assigned there
disabled: boolean;         -> disable textbox
 dateOptions: {
	defaultStartDate: Date || String,    -> default start date
	minStartDate: Date || String, ->min date
	maxStartDate: Date || String, -> max date
}

**Code Example:**

    //ts
    filterData = [
		{
		title:  'Choose Date',
		type:  ZenFilterTypes.DATEPICKER,
		keyToEmit:  'date',
		disabled:  false,
		dateOptions: {
			defaultStartDate: new  Date(),
			maxStartDate: '01-10-2025',
			minStartDate: '01-02-2025'
		}
	]
	//html
	<zen-filter  [filterDetails]="filterData" (onFilterChange)="onFilterChange($event)"></zen-filter>
